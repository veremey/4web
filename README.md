Для начала установить [Bower](http://bower.io/) и [Gulp](http://gulpjs.com/) глобально.

Установить локально gulp и bower выполнив следующие команды:

1. *npm install gulp*

2. *npm install bower*

Установить плагины

1. *npm install*

2. *bower install*
